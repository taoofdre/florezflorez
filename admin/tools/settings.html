Stripe

Set in your Vercel project under Settings → Environment Variables. Your secret key is in the Stripe dashboard under Developers → API Keys.

STRIPE_SECRET_KEY Server-side key used to create checkout sessions. Starts with sk_live_ (production) or sk_test_ (test mode).
Stripe API Keys → Vercel Dashboard →

GitHub OAuth

Powers the CMS and admin login. Set in Vercel under Settings → Environment Variables. Create or manage your OAuth app in GitHub developer settings.

GITHUB_CLIENT_ID OAuth App client ID. Public identifier for your GitHub OAuth app.
GITHUB_CLIENT_SECRET OAuth App client secret. Keep this private — never commit it to your repository.
GitHub OAuth Apps → Vercel Dashboard →
